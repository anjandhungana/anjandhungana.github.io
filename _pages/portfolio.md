## Research Projects

### Poultry Disease Detection
Deep learning system for automatic disease detection from chicken manure images using convolutional neural networks.
- **Technologies**: Python, TensorFlow, Computer Vision
- **Timeline**: 2024-Present
- **Status**: In Progress

### IoT Poultry Monitoring System
Real-time monitoring system for poultry health and behavior using distributed IoT sensors and cloud-based analytics.
- **Technologies**: IoT, Sensor Networks, Python, RESTful APIs
- **Timeline**: 2024-Present
- **Impact**: Enables predictive health monitoring and early disease detection

### Metabolomics Analysis of Livestock
Comprehensive analysis of livestock metabolism patterns and methane production using advanced bioinformatics tools.
- **Technologies**: MetaboAnalyst, Statistical Analysis, R
- **Timeline**: 2022-2024
- **Publications**: Multiple peer-reviewed papers (see Publications)

### Poultry Behavior Tracking System
Computer vision-based system for tracking and analyzing poultry behavior patterns for welfare assessment.
- **Technologies**: Computer Vision, Deep Learning, Motion Analysis
- **Timeline**: 2024-Present
- **Applications**: Welfare monitoring, behavioral analysis, productivity metrics
