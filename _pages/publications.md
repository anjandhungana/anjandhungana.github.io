## How to Add Publications

To add publications to this page, create markdown files in the `_publications/` directory with the following YAML front matter:

```yaml
---
title: "Publication Title"
authors: "Author Names"
date: 2024-01-15
venue: "Journal or Conference Name"
paperurl: "https://link-to-paper"
citation: "Your citation here"
---
```

Each publication will be automatically added to this list.
