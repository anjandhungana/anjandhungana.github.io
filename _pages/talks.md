## How to Add Talks

To add talks and presentations, create markdown files in the `_talks/` directory with the following format:

```yaml
---
title: "Talk Title"
collection: talks
type: "Conference Talk"
permalink: /talks/2024-01-01-talk-title
venue: "Conference Name"
date: 2024-01-01
location: "City, Country"
---
```

Talks will be automatically compiled on this page and in your CV.
