## Blog Posts

Coming soon! Add blog posts to the `_posts/` directory to have them automatically appear here.

### How to Add Posts

Create markdown files in the `_posts/` directory with the naming convention: `YYYY-MM-DD-post-title.md`

Use the following YAML front matter:

```yaml
---
title: "Post Title"
date: 2024-01-15
categories: research
tags: [ai, IoT, livestock]
excerpt: "Brief description of your post"
---
```

Posts will be automatically listed here, sorted by date.
